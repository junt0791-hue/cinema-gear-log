Cinema Gear Log
Project Overview
A web-based digital archive for filmmakers and enthusiasts to categorize, explore, and study 48 iconic motion picture cameras. The application features a comprehensive library spanning classic film cameras to modern digital cinema systems.
Core Features
 Brand-Model Hierarchy: Organized classification for efficient navigation.
 Detailed Archive: Dedicated profiles for every camera model, including technical types (Film vs. Digital).
 Instant Search: Powerful real-time filtering to quickly locate any camera in the collection.
Technical Stack
 Built with HTML5, CSS3, and JavaScript for a responsive, clean, and professional user interface.
